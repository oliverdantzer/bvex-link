◆ make_requester()

Requester make_requester ( char * metric_id,
char * node,
char * service )

Creates a requester. Used by functions like request_int to specify endpoint and metric ID to request a value from. Threading multiple requests to the same Requester is not recommended.

Parameters
metric_idThe metric ID to request.
nodeThe node to connect to.
serviceThe service to connect to.
Returns
Requester structure.

This requester object is used instead of requesting a metric directly from a socket file descriptor to make it difficult to request different metric ids from the same socket.

Example usage:

Requester altitudeRequester = make_requester(
"altitude", "localhost", "3000");
if (altitudeRequester.socket_fd < 0) {
// handle error
} else {
// use altitudeRequester
}

RequestDoubleResult request_double ( Requester * reqr)

Sends a request for a double value sample from the metric id of reqr from the server reqr.socket_fd is connected to. This function is blocking and will wait for the send operation to complete and for the response to be received.
